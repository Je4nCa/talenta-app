# UI Guidelines

# Design Style

Inspirations:
- Apple Wallet
- Copilot Money
- Linear
- Notion
- Monarch Money

---

# UI Feel

The app must feel:
- premium
- modern
- minimal
- smooth
- elegant

---

# Main UI Rules

## Mobile First
Design for iPhone first.

## Quick Actions
Main actions should require minimal taps.

## Financial Clarity
Financial information must be extremely readable.

---

# Category System

Use:
- emojis
- colors
- icons

Examples:
- 🍔 Food
- 🏠 Apartment
- ☕ Coffee
- 🛍️ Shopping

---

# Animations

Use Framer Motion.

Rules:
- subtle
- smooth
- fast
- non-intrusive

---

# Dashboard Rules

Dashboards should prioritize:
- clarity
- hierarchy
- summaries
- visual simplicity

---

# Dark Mode

Dark mode by default.

Style:
- soft blacks
- muted grays
- accent colors
